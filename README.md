## Demo

You'll need Docker to be able to see the app

```bash
cp ./nestjs/.env.example ./nestjs/.env

docker compose -f ./nestjs/docker-compose.yml up -d
```
See [http://localhost:8080](http://localhost:8080) in your browser.
